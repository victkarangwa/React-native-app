# React Native App
 This is a simple react native app which demonstrates the basics for React-native

## Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

   * [Expo-cli](https://www.npmjs.com/package/expo-cli)
   * Expo client for [Android](https://play.google.com/store/apps/details?id=host.exp.exponent)/[iOS](https://itunes.com/apps/exponent)
